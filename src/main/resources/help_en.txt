[Specifying Parameters]

--help
    Display the method of specifying parameters

--url
    Specify the API URL (default is "https://api.openai.com/v1/chat/completions")

--model
    Specify the gpt model (default is "gpt-4")

--apikey
    Specify the API key (if not specified, it is retrieved from the environment variable "OPENAI_API_KEY")

--resource
    Specify the type of resource to input (default is "others")
    Available types are:
    java : Java source code (targets files with ".java" extension)
    js : JavaScript or TypeScript source code (targets files with ".js" or ".ts" extensions).
    python : Python source code (targets files with ".py" extension)
    sql : SQL code (targets files with ".sql" extension)
    page : Page files (targets files with extensions "page", ".html", ".htm", ".xhtml", ".jsp")
    shell : Shell script (targets files with ".sh", ".bash", ".ksh", ".bash" extensions)
    others : General resources other than the above

--gen
    Specify what you want to generate from the resource (cannot specify both "--gen" and "--gen-table")
    Available types are:
    spec : Specification
    summary : Summary
    review : Review results and improvement suggestions
    others : General output other than the above (default if neither "--gen" nor "--gen-table" is specified)

--gen-table
    If you want to generate a list from the resource, specify its name (cannot specify both "--gen" and "--gen-table")

--fields
    If generating a list from a resource, enumerate the column names (mandatory for "--gen-table")

--output-scale
    Specify the approximate size of the output (default is "nolimit")
    small : Small size (about 50 characters)
    medium : Medium size (about 200 characters)
    large : Large size (about 500 characters)
    nolimit : No limit

--src
    Specify the file or directory of the source to be read (mandatory)
    If a directory is specified, files are processed recursively.
    If a ZIP file is specified, the contents are processed recursively.

--regex
    Specify a regular expression string to filter the sources to be read (optional)

--dest
    Specify the full path of the directory or file name of the markdown file to output.
    If a directory name is specified, the default file name is "brain2doc-{resource name}-{output type name}-{yMMddHHmmss}.md"
    (e.g. "brain2doc-java-review-20230924104914.md")
    If this option is not specified, it will be output in the same directory as the source file with the default file name.

--lang
    Specify the input and output language (default is Japanese)
    ja : Japanese
    en : English

--template
    Specify the full path of any template file (optional)
    Only the differences from the default template file are overwritten.

--proxyURL
    Specify the proxy server URL when calling the API.

--connect-timeout
    Specify the connection timeout time in seconds when calling the API (default is 10 seconds)

--timeout
    Specify the read timeout time in seconds when calling the API (default is 300 seconds)

--retry-count
    Specify the maximum number of retries when a read timeout occurs during an API call (default is 3 times)

--retry-interval
    Specify the retry interval in seconds when a read timeout occurs during an API call (default is 60 seconds)

--auto-split
    If this option is specified, when a single API call reaches the token limit, it automatically switches to a mode that splits into appropriate sizes and resends.
    If not specified, the source will be skipped.
